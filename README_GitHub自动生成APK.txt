GitHub 自动生成 APK 方法：

1. 打开 https://github.com/new 新建一个仓库，比如 DepressionTestApp
2. 把本文件夹 DepressionTestApp 里的所有文件上传到仓库根目录
3. 上传完成后，点仓库上方 Actions
4. 左侧选择 Build APK
5. 点 Run workflow
6. 等运行结束后，进入本次运行记录
7. 页面底部 Artifacts 里下载 depression-test-50-debug-apk
8. 解压后得到 app-debug.apk，发到安卓手机安装即可

如果手机提示“未知来源应用”，在手机设置里允许浏览器/文件管理器安装未知应用即可。
