这是“抑郁情绪自测50题”Android 项目源码。

当前项目功能：
1. 50道题，每题0-3分。
2. 总分0-150分，自动输出五个等级。
3. 第49/50题为安全预警题，得分较高时自动显示危机提示。
4. 单机离线运行，不上传用户答案。

打包APK：
1. 安装 Android Studio。
2. 打开 DepressionTestApp 文件夹。
3. 等待 Gradle 同步完成。
4. 点击 Build > Build Bundle(s) / APK(s) > Build APK(s)。
5. APK通常生成在 app/build/outputs/apk/debug/app-debug.apk。

注意：本工具只做情绪筛查，不能替代医生诊断。
